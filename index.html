PROJECT DETAIL:

To assist the stock investor to make better informed decisions, a complex application has been developed to generate analytical tables and diagrams to summarize and visualize the performance of a particular stock or index. First, the Stock Quote selection will provide up-to-date stock or index information. Second, for a faster stock prediction, the Kalman Filter Prediction uses the fast Kalman Filter machine learning algorithm to predict tomorrow's stock momentum, prices, and volume. For a slower prediction, the Stock Forecast selection uses a variety of machine learning algorithms such as Random Forest, Nearest Neighbor, Neural Network, SVM, Naive Bayes, Kalman Filter, Ada Boost, and etc to predict tomorrow's stock momentum, prices, and volume in a majority voting system in order to get the best results. Moreover, there are 7 analytical stock analysis tools such as MACD, SMA, OBV, RSI, AROON, and etc to help the investor to visualize the future performance of the stock or index.

Finally, the Stock Summary selection will summarize the prediction and analysis information in a detailed table and a visualized diagram of that particular stock or index.

USER INTERFACE:

The application takes the stock or index symbol from the user such as Google(GOOG) or Facebook(FB). Next, the user has to select one of the following choices from the menu:

Stock Quote
Stock Moving Average Convergence Divergence
Stock Simple Moving Averages
Stock On Balance Volume
Stock Accumulation Distribution Line
Stock Average Directional Index
Stock Aroon Indicator
Stock Relative Strength Index
Stock Stochastic Oscillator
Stock Kalman Filter Prediction
Stock Forecast
Stock Analysis Summary

Finally, the user can select the number of days to analyze or predict a particular stock or index, ranging from 55 days to 2,600 days in a slider.
